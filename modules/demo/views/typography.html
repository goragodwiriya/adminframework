Paragraphs

นี่คือย่อหน้าที่เกิดจาก tag <p>....</p> คุณสามารถใส่เนื้อหาต่างๆได้ภายใต้ tag นี้

จัดตัวอักษรชิดขวาด้วย <p class=right>...</p>

จัดตัวอักษรกึ่งกลางด้วย <p class=center>...</p>

จัดตัวอักษรชิดซ้ายด้วย <p class=left>...</p>

Code/Blockquote

ใช้สำหรับระบุส่วนที่เป็นโค้ด โดยอยู่ภายใต้ tag <code>....</code>
ใช้สำหรับการอ้างอิงไปยังคำพูด <blockquote>...<blockquote>

Inline Text

class term0-term10 สำหรับการทำ Highlight ข้อความ .term0 .term1 .term2 .term3 .term4 .term5 .term6 .term7 .term8 .term9 .term10 .term11

tag <em> ใช้สำหรับเน้นข้อความสำคัญ

tag <strong> และ <b> และ <dt> แสดงผลเป็นตัวหนา

tag <i> และ <dfn> แสดงผลเป็นตัวเอียง

tag <ins> และ <u> แสดงผลเป็นตัวขีดเส้นใต้

tag <del> แสดงผลเป็นตัวขีดฆ่า

tag <acronym> และ <abbr>

tag <sub> และ <sup> แสดงผลเป็นตัวห้อยและตัวยก

tag <small> และ <big> สำหรับปรับขนาดตัวอักษร
